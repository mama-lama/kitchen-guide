Проект: “Кухонный помощник”

Автор проекта: Ксения Ширяева

Идея: Идея была в том, чтобы создать улучшенную версию обычной бумажной книги, в которую пользователь мог вносить бы 
свои рецепты или секреты готовки. Чтобы в книге можно было легко найти рецепт по поиску. 
Реализация: Весь код программы реализован в файле “main.py”. Файл состоит из 9 классов, из них 7 основные
(стартовое окно, окно поиска и выбора рецепта, окно для просмотра рецепта, окно добавления, справочник с поиском, 
добавление справки и окно для просмотра информации). Изображения сохраняются в файл “imgs”, а рецепты хранятся в БД “Progect”.

Фишечки:
•Можно искать блюда по введённым ингредиентам.
•Рецепты и справки можно открывать двойным щелчком мыши или нажатием Enter
•Можете добавлять свои рецепты и секреты готовки
•К каждому рецепту можно добавить фото

Возможные улучшения:
•Расширить поиск по статьям
•Попробовать настроить систему авторизации, чтобы можно было создать список избранных, приготовленных и нелюбимых блюд, 
в который пользователь мог бы добавлять рецепты
